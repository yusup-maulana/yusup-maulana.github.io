Cashflow Proyek adalah aplikasi sederhana untuk membantu estimasi arus jumlah uang yang mengalir keluar dari rencana dengan aktual seiring berjalannya suatu item pekerjaan dengan persentase tertentu

Demo : Klik disini

Fitur unggulan:

  • Perbandingan aktual dengan rencana awal, ketika aktual tidak sesuai dengan rencana, aplikasi akan memberikan indikasi warna pada persentase sehingga mudah di kenali
  • Perhitungan margin per item pekerjaan sampai seluruh proyek
  • Pengelompokan item pekerjaan

Bahan pembuatan

  • HTML sebagai kerangka
  • PHP sebagai bahasa program
  • CSS sebagai pemanis dan layout
  • Javascript dan Jquery sebagai interaksi langsung
  • MYSQL sebagai database
