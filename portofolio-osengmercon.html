Aplikasi ini berfungsi untuk mengelola restock bahan makananan sebelum jadi, sehingga pengelola dapat dengan mudah memonitor dan menjaga kesetabilan stock bahan yang disupply kepada outlet-outlet/cabang lain.

Demo : Klik disini
pengelola
user : admin
pass : admin


outlet ++
user : outlet1
pass : admin

Fitur unggulan:

  • Multi outlet (bisamembuka lebih dari 1000 cabang baru)
  • terdapat 1 sisi pengelola utama, dan multi admin outlet
  • Pembukuan stok, outlet
  • Riwayat pengajuan otulet/cabang
  • Riwayat order
  • pengelolaan stok
  • submit stok
  • stok outlet
  • laporan lengkap
  • dilengkapi chart

Bahan pembuatan

  • HTML sebagai kerangka
  • PHP sebagai bahasa program
  • CSS sebagai pemanis dan layout
  • Javascript dan Jquery sebagai interaksi langsung
  • MYSQL sebagai database
