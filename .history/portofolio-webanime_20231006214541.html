Demo : Klik di sini Template web didesain khusus untuk layanan video dari pihak ketiga. Untuk menonjolkan kecepatan tanpa memerlukan reload/refresh browser, template ini sengaja mengusung kesan simpel, minimalis, sekaligus kreatif.

Di halaman depan, langsung disuguhi tampilan full-width slider yang elegan. Pada bagian itulah, Anda bisa memasang sejumlah video unggulan.

Ketika di scroll, Anda akan menyadari detail tampilan yang apik. Background web disusun dari sekumpulan gradiasi yang tidak sakit mata. Ilustrasi gambar di web dibuat dalam format kotak. Begitu pun dengan judul post yang dibingkai berwarna gelap dengan kilauan mencolok.

Anda bisa menambahkan banyak kategori dengan berdasarlan tahun, genre, musim dan studio,

Admin page : Klik disini

Fitur unggulan:

  • Full-width slider preview konten dari resolusi gambar HD
  • Moblie bisa diakses dengan baik lewat handphone,tablet,Laptop maupun PC
  • tanpa icons mempercepat pengunjung mengaksesa
  • Hover effects mempercantik tampilan web sekaligus memudahkan link agar dikenali
  • Cocok untuk: usaha layanan web video pihak ketiga
  • Single Page admin untuk pengelolaan link yang presisi, yang sangat user friendly

Bahan pembuatan

  • HTML sebagai kerangka
  • PHP sebagai bahasa program
  • CSS sebagai pemanis dan layout
  • Javascript dan Jquery sebagai interaksi langsung
  • MYSQL sebagai database
