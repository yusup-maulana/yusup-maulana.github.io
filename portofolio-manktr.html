Man. KTR adalah aplikasi manajemen kantor yang dibuat untuk menunjang kebutuhan pengelolaan data di PT. Raya Azura Persada, dengan fitur-fitur sebagai berikut :

  • Kelola Data Admin
  • Kelola Data Karyawan
  • Kelola Data Barang/Material beserta fluktuatifnya
  • Kelola Data Suplayer
  • Kelola Data Proyek da estimator
  • Kelola Data Purchase Order
  • Estimasi Control
  • Laporan pertanggung jawaban
  • dilengkapi dengan grafik
Aplikasi ini terdapat 3 sisi user untuk menjalankan nya
yang pertama admin 1 sebagai admin utama untuk melakukan kelola admin, membuat Purchase Order, Mengelola barang, Pegawai Suplayer, dan pengiriman data pertanggung jawaban yang nantinya akan diteruskan ke admin2. Adapun user sama password nya untuk user 1 :
User : admin1
Password : admin1

Kemudian di sisi user 2 tugas nya untuk memvalidasi Pruchase order dan laporan pertanggung jawaban yang dikirimkan admin1. di admin 2 ini terdapat fitur acounting debit,kredit saldo, dan pengolaan transaksi,
User : admin2
password : admin2

lalu terakhir user 3, bisa dikelola oleh estimator
user : admin3
password : admin3
